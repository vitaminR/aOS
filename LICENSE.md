# 6.aOS Licensing Model

This repository uses a **dual-license structure** to separate the public reference model from software implementation assets.

## A) Public Reference Model Materials — CC BY 4.0

License: **Creative Commons Attribution 4.0 International (CC BY 4.0)**  
Canonical text: <https://creativecommons.org/licenses/by/4.0/>

Applies to:

- `01.Reference-Model/agentic-reference-stack-v1.md`
- `00.Governance/standardization-pack/`
- `03.Research/`

Intent:

- Allow use, sharing, and adaptation with attribution
- Support neutral standards-track credibility and reuse
- Keep the model standards-friendly rather than proprietary-only

Attribution baseline:

- Credit `{a}OS` / original author in derivative or redistributed model artifacts
- Preserve reference to version where practical (e.g., v1.0)

## B) Explorer Software — Apache License 2.0

License: **Apache License 2.0**  
Canonical text: <https://www.apache.org/licenses/LICENSE-2.0>

Applies to software source in:

- `../90.aOS-Explorer/` (including `prototype.html`, `stack-mapper.html`, `docs.html`, `validate.py`, and subsequent application code)

Intent:

- Enterprise-friendly open-source software terms
- Explicit patent grant and patent retaliation protections
- Practical reuse for implementation and tooling ecosystems

## C) Atlas-Specific License Addendum

A project-specific commercial/internal-use addendum is defined at:

- `00.Governance/ATLAS_LICENSE_ADDENDUM.md`

This addendum governs Atlas internal rights and boundaries while preserving author ownership and external commercialization rights.

## Scope and precedence

1. If content is part of the reference model/governance/research materials listed in section A, apply **CC BY 4.0**.
2. If content is software/tooling source listed in section B, apply **Apache 2.0**.
3. For Atlas internal deployment rights and restrictions, apply section C addendum terms.

## Notes

- `{a}OS` remains the product/brand name.
- Neutral standards-track title remains: **Agentic Systems — Reference Model and Vocabulary**.
- The v1.0 model vocabulary and strata are locked unless version governance explicitly changes them.
