# AFS12 — Visual Direction: "A Framework for Every AI Platform Decision"

## Plot Options

### Plot A — Triple Heatmap (Side-by-Side Columns)

Three columns, each representing a platform: Azure AI Foundry, AWS Bedrock, "Internal Platform." Each column shows the 7-stratum model as stacked rows with coverage indicators:

- L1-L3: Green/strong on all three
- L4: Yellow/partial on all three
- L5-L7: Red/absent-to-weak on all three

Label: "{a}OS — Same Pattern, Every Platform"

**Pros:** Directly proves the universality claim. Clean, data-driven. Scannable at mobile size.
**Cons:** Three columns may feel crowded on LinkedIn mobile. Requires legible labels at small size.

### Plot B — Convergence Diagram (Recommended)

Three separate platform icons at the top (Azure diamond, AWS arrow, generic building for "Internal") each with a downward arrow converging into a single {a}OS 7-stratum stack at center. The stack shows the universal pattern: green bottom, red top. The convergence visual communicates: different platforms, same structural gap.

Label top: "Different platforms"
Label bottom: "Same gap"
Subtitle: "{a}OS 7-Stratum Model"

**Pros:** Tells the story of the series in one image — started with one platform, ended with a universal framework. Strong visual narrative. Works at mobile size.
**Cons:** Requires careful layout to avoid clutter.

### Plot C — Before/After Arc

Left side: Post 1's "gap diagram" (the platform stack with missing top). Right side: The same stack, but now with {a}OS labels on each stratum and coverage indicators. An arrow between them labeled "12 posts later."

Label left: "Post 1 — unnamed gap"
Label right: "Post 12 — named strata"

**Pros:** Rewards series readers with a visual callback. Closes the arc cleanly.
**Cons:** Less impactful for readers who did not follow the series. Relies on recognition of Post 1 visual.

## Recommended

**Plot B** — The convergence diagram tells the capstone story in one image: multiple platforms, one structural pattern, one framework. It works for both series readers and new viewers. The visual metaphor (convergence into a single lens) matches the post's thesis directly.

## Art Direction Notes

- Three platform representations should be recognizable but not use official logos (avoid trademark issues). Use geometric shapes or abbreviated text: "Azure," "AWS," "Internal."
- {a}OS 7-stratum stack uses the canonical layer names and orange {a}OS brand accent
- Dark or neutral background, consistent with series visual style
- Sans-serif typography
- Must work at LinkedIn mobile size (1200x627 or 1080x1080)
- The convergence should feel clarifying, not reductive — the framework reveals structure, it does not oversimplify
